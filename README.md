# Bank Customer Churn Analysis using Python

## Project Overview
This project focuses on analyzing bank customer churn data to identify patterns and factors that influence customer retention. The analysis was performed using Python libraries for data cleaning, visualization, and exploratory data analysis (EDA).

The main goal of this project is to understand customer behavior and generate meaningful business insights from banking datasets.

---

## Objectives
- Analyze customer churn behavior
- Identify factors affecting customer retention
- Perform data cleaning and preprocessing
- Create visualizations for better insights
- Explore customer demographics and financial patterns

---

## Dataset Used
The project uses banking customer datasets including:
- BankChurners.csv
- Churn_Modelling.csv

> Note: Large dataset files were excluded due to GitHub file size limitations.

## Key Insights
- Customers with lower activity are more likely to churn
- Age and account balance affect customer retention
- Certain customer segments show higher churn rates
- Credit-related factors influence customer behavior
